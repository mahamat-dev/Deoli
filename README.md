# Deoli

Deoli is a clean, focused iOS reminder app designed to help you capture tasks fast, prioritize them, and stay on top of what matters.

![Deoli hero](assets/deoli-hero.svg)

## Overview

Deoli organizes reminders by priority and category, supports due dates, and keeps completed items neatly separated so your day stays clear.

## Features

- Quick add for new reminders with title, notes, priority, category, and due date
- Category chips for fast filtering
- Priority badges and completion states
- Edit and delete with context actions and swipe gestures
- Clean, modern SwiftUI layout

## Project Structure

- Deoli: App source
- DeoliTests: Unit tests
- DeoliUITests: UI tests
- Deoli.xcodeproj: Xcode project

## Build

Open Deoli.xcodeproj in Xcode and run the Deoli scheme.
